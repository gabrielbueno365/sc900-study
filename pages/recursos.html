📄 Cheat Sheets e Quick References

🛡️ Zero Trust Cheat Sheet

3 Core Principles:

  • Verify explicitly: Always authenticate based on all available data
  • Least privilege: Limit access with JIT/JEA, risk-based adaptive policies
  • Assume breach: Minimize blast radius, verify encryption, analytics

6 Foundational Elements:

  • Identities | Devices | Applications | Data | Infrastructure | Networks

🔐 Conditional Access Cheat Sheet

IF Conditions:

  • Users: All users, select users, groups, roles
  • Cloud apps: All apps, select apps, user actions
  • Conditions: Sign-in risk, device platforms, locations, device state

THEN Actions:

  • Grant: Require MFA, compliant device, hybrid joined device
  • Block: Block access completely

⚔️ Defender XDR Cheat Sheet

Core Products:

  • Defender for Endpoint: Device/workload protection
  • Defender for Office 365: Email, SharePoint, Teams
  • Defender for Identity: On-premises AD protection
  • Defender for Cloud Apps: CASB solution

Integration Benefits:

  • Unified incidents | Attack story | Automated investigation

📊 Compliance Solutions Cheat Sheet

Key Differences:

  • Sensitivity Labels: CLASSIFY and protect data
  • DLP Policies: PREVENT data loss and sharing
  • Retention Policies: KEEP or DELETE content over time
  • Records Management: IMMUTABLE regulatory records

Cloud Security:

  • CSPM: Configuration + Posture Management
  • CWPP: Workload Protection Platform
